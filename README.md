## :nut_and_bolt: Development


#### 1. Clone the Repository

```Bash
git clone https://github.com/nightwarriorftw/adopt-me.git
cd adopt-me
```

#### 2. Install the dependencies

```BASH
npm install
```

#### 3. Run server:

```BASH
npm run dev
```
